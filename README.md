## 4. Miniguia de Estudo (Entrega Final) 📖

### 💡 Resumo Estruturado: O Ambiente de Descoberta Autônoma
A análise de dados real não é sobre saber o código, mas sobre saber onde encontrar a resposta.

*   **Filosofia:** O programador independente usa o IPython como painel de controle.
*   **Técnicas de Exploração:** Uso de `?` para docstrings e `??` para inspeção de código fonte.
*   **Performance:** Benchmarking com `%timeit` para evitar códigos lentos.

### 📜 Glossário de Conceitos Chave
*   **Vetorização:** Técnica de eliminar loops `for` usando arrays NumPy (Legado de Travis Oliphant).
*   **Cicatrizes Técnicas:** A experiência acumulada em falhas reais de sistemas que cursos teóricos não ensinam.
*   **Magic Commands:** Atalhos do IPython (`%`) que resolvem gargalos de produtividade.

### 🧪 Prompts Reutilizáveis para Revisão
*   *"Aja como Jake VanderPlas e explique o conceito de vetorização neste bloco de código: [COLE SEU CÓDIGO]"*
*   *"Analise este erro de traceback usando a lógica de %xmode verbose e me ajude a encontrar o erro de lógica."*
